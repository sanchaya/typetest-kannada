
## How to add a new section

1. Add a key to `testSet` in `index.html`:
   ```js
   mySection: "ಯಾವುದಾದರೂ ಪಠ್ಯ ಇಲ್ಲಿ",
   ```
2. Add an HTML block in the render area:
   ```html
   <div class="section">
     <div class="section-meta">
       <span class="section-tag">11 · My Section</span>
       <span class="section-desc">Short description</span>
     </div>
     <div class="test-text" id="render-mySection"></div>
   </div>
   ```
3. Add one line in `renderContent()`:
   ```js
   $('render-mySection').textContent = testSet.mySection;
   ```
